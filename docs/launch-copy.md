### Title

Hermes Relay for Chrome

### Short description

Browser extension for Hermes Agent by Nous Research with page context, continuity, tracked pages, and AI handoff.

### Long description

Hermes Relay for Chrome is the browser companion for Hermes Agent by Nous Research.

It gives Hermes a direct line into the pages you read, revisit, track, and hand off into AI workflows. Instead of treating the browser like a one-shot prompt surface, Hermes Relay helps Hermes keep continuity around the pages that matter.

Use Hermes Relay to:

- understand the current page with Hermes
- keep notes, snapshots, and tracked-page state
- revisit a page with continuity
- save durable facts, preferences, and workflows into Hermes memory
- build context bundles for Claude, ChatGPT, and Gemini
- insert Hermes context directly into supported chat inputs

Hermes Relay works with a local Hermes Agent API server and is designed for people who want Hermes to feel present in the browser, not bolted on after the fact.

This first public launch is intended for GitHub: Hermes Agent maintainers, early Hermes users, and contributors who are comfortable loading an unpacked Chrome extension or release zip.
