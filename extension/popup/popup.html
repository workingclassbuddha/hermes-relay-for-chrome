Privacy & Safety

Page data is redacted before it is sent, and only when you click an action or create a watcher.
Browser actions require approval in the workspace before Hermes can click, fill, or open pages.
Memory actions return a receipt and should never silently store page content.
AI-site insertion uses explicit site permission and your active browser session.
